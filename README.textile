h1. ServiceMix IDE

This project provides support for multiple configuration files related witch
ServiceMix deployment.

h2. Feature mix

This is only planned feature mix, currently only part of this list is implemented.

Support for custom XML namespaces:
* ServiceMix components
* ActiveMQ
* Camel
* CXF

Integrated content assistant for elements listed above
* Camel
* ServiceMix components
* Karaf blueprint extensions 

h2. Requirements

This plugin requires:
* Eclipse 3.5 
* Spring IDE
* M2Eclipse

h2. Building

We use Apache Maven *3.0* with "Tycho":http://tycho.sonatype.org.
Make sure you use correct Maven version. 

h3. Remote resources

Some of resources we using are remote - we download it and put into version control
system to avoid future location change problems.

Commands that can be helpful:
??wget -nd -np -r -e robots=off -U "Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.6) Gecko/20070725 Firefox/2.0.0.6" http://repo1.maven.org/maven2/org/apache/servicemix/servicemix-core/ -Axsd??
Downloads all xml schema files from servicemix-core directory and sub-directories. Files are stored without following remote directories.
